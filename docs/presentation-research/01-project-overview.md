# 01 - نظرة عامة على المشروع
> الحالة: ✅ مكتمل

---

## ملخص (للـ Presentation)

**Code Mentor** هو منصة تعليمية ذكية تعمل بالذكاء الاصطناعي، تسد الفجوة بين "المعرفة الأساسية بالبرمجة" و"الكفاءة المهنية في هندسة البرمجيات" للمطورين العصاميين، ومغيّري المسار الوظيفي، وطلاب الجامعات.

**الجملة المفتاحية:** "Senior-developer-equivalent code feedback in ≤5 minutes per submission"

---

## المشكلة (Problem Statement)

### الفجوة الأساسية
المتعلمون الجدد في البرمجة يواجهون فجوة واضحة بين:
- ✅ **ما يمتلكونه**: معرفة أساسية بكتابة الكود (من الدورات المجانية، YouTube، الجامعة)
- ❌ **ما ينقصهم**: تقييم احترافي لجودة الكود، مراجعة أمنية، تصميم معماري — أي ما يحصل عليه المطور فقط من مطور أقدم (Senior Developer)

### المشكلة بالتحديد
1. **غياب الـ Feedback المتخصص**: المتعلمون يكتبون كود يعمل لكنهم لا يعرفون هل هو "جيد بما يكفي" للوظيفة
2. **تكلفة الحصول على Mentor**: المعسكرات التدريبية (Bootcamps) تكلف $10,000+
3. **الشهادات لا تثبت الكفاءة**: إتمام دورة لا يعني القدرة على كتابة كود إنتاجي
4. **ضعف التقييم الأكاديمي**: ملاحظات المعيدين على المشاريع متأخرة وسطحية

## الحل المقترح (Proposed Solution)

### الفكرة الأساسية
منصة تعليمية متكاملة تقدم:

1. **تقييم تكيّفي (Adaptive Assessment)** — يفهم مستوى المتعلم الحقيقي باستخدام خوارزمية IRT
2. **مسار تعلم مخصص (Personalized Learning Path)** — يُنشأ بالذكاء الاصطناعي بناءً على نقاط الضعف
3. **مراجعة كود متعددة الطبقات (Multi-Layered Code Review)** — تجمع بين:
   - 6 أدوات تحليل ثابت (Static Analysis): ESLint, Bandit, Cppcheck, PHPStan, PMD, Roslyn
   - مراجعة بالذكاء الاصطناعي عبر 5 محاور: الصحة، القراءة، الأمان، الأداء، التصميم
4. **سيرة ذاتية تعليمية (Learning CV)** — قابلة للمشاركة وتثبت المهارات بالبيانات

### الجملة الملخصة
> "Code Mentor gives self-taught developers the kind of feedback they'd otherwise only get from a senior mentor — at a fraction of the cost."

### الـ Core Loop (الدورة الأساسية)
```
Assessment → Learning Path → Submit Code → AI Review → Feedback → Improve → Resubmit
```

## المنافسون والحلول الموجودة

| المنافس | ما يقدمه | ما ينقصه |
|---------|----------|----------|
| **LeetCode** | تمارين خوارزمية + اختبار صحة الحل | لا يقيّم جودة الكود أو التصميم أو الأمان — يختبر correctness فقط |
| **YouTube / دورات مجانية** | شرح المفاهيم | لا تقيّم المشاريع الفعلية للمتعلم |
| **Bootcamps** | تعليم مكثف + مراجعة من مدربين | تكلفة عالية جداً ($10k+) |
| **SonarQube** | تحليل كود ثابت | أداة للمطورين المحترفين، لا تقدم مسار تعلم أو تقييم شامل |
| **GitHub Copilot** | كتابة كود ذكي | يساعد في الكتابة لكن لا يعلّم أو يقيّم |
| **الجامعة** | نظريات + مشاريع | ملاحظات المعيدين سطحية ومتأخرة، لا تغطي جودة الكود الإنتاجي |

## ما يميز Code Mentor (Unique Value Proposition)

### 1. مراجعة كود بمستوى مطور أقدم في ≤5 دقائق
- 6 أدوات تحليل ثابت + مراجعة LLM = تقرير شامل من 1-3 صفحات
- تعليقات على كل سطر (inline annotations)

### 2. نظام تعلم تكيّفي بالكامل
- تقييم بخوارزمية 2PL IRT (ليس قواعد بسيطة)
- مسار تعلم يُنشأ بالـ AI ويتكيّف مع تقدمك
- إعادة ترتيب وتبديل المهام بناءً على أدائك

### 3. Multi-Agent Code Review
- 3 وكلاء متخصصين (أمان / أداء / معمارية) يعملون بالتوازي
- نتائج أعمق من مراجعة واحدة

### 4. RAG Mentor Chat
- شات ذكي يفهم الكود المرفوع فعلياً (ليس نصائح عامة)
- مبني على Retrieval-Augmented Generation

### 5. Learning CV
- سيرة ذاتية تعليمية قابلة للمشاركة
- تثبت المهارات بالبيانات وليس بشهادات إتمام

### 6. Project Audit
- فحص شامل لمشروع كامل (ليس task واحد)
- تقرير من 8 أقسام بما فيه Completeness Analysis
